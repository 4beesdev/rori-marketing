# Rori Marketing

Meta Marketing API integration for Rori.

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env
# Add your META_ACCESS_TOKEN to .env
```

## Usage

```bash
python src/meta_api.py
```

## GitHub Secrets

| Secret | Description |
|--------|-------------|
| `META_ACCESS_TOKEN` | Meta Graph API access token |
